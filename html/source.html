Data sources

Data is the backbone of any data analytics business. The data is an unstructured set of facts and figures from various sources. Data sources can vary depending on what the research needs. Data analysis and interpretation is solely based on different types of data from its sources. Performs data collection work to gather information.

In short, data sources are physical or digital places where information is stored in a spreadsheet, data object, or other storage format. Data can be collected from two places: internal and external sources. Information collected from internal sources is called “primary data,” while information collected from external references is called “secondary data.”

There were many data sources used in the creation of this site. At the beginning, shape files were created for health services in North Sinai Governorate, which are hospitals and ambulances, and added them to the ArcGIS Online site.
